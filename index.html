Características

  • Framework
  • Gratuito
  • Abierto
  • Apache cordova
  • Creación de aplicaciones nativas para móviles

¿ Cómo funciona PHONEGAP ?

Phonegap nos proporciona una API que nos permite acceder a las funcionalidades nativas de los dispositivos móviles utlizando javascript

Funcionalidades de la API de PHONEGAP

Aplicaciones cliente servidor

  • La aplicacion cliente de PhoneGap se comunica con una aplicación en el servidor para recibir/enviar datos.
  • El servidor suele ser un servidor web que sirve una aplicacion escrita en un lenguajes como Java, PHP, Ruby,etc…
  • La base de datos es gestionada a traves de una aplicación en el servidor. La comunicacion con el servidor se suele basar en peticiones HTTP estandar para contenido HTML, como REST, XML, JSON, SOAP.
  • Serian exactamente las misma técnicas que utilizarias en una aplicación basada en AJAX.

Arquitectura de la aplicación

  • Modelo de página única
  • La página permanece cargada en memoria
  • Los datos se muestran actualizando el DOM HTML
  • Los datos se guardan desde la aplicacion en el servidor a traves de técnicas AJAX
  • Las variables se mantienen en memoria a traves de Javascript

Instalación Android

  • Creamos un proyecto android con el wizard de eclipse
  • Añadimos al proyecto los directorios /libs y assets/www
  • Copiar cordova-x.x.x.jar al directorio libs (desde el directorio donde instalamos el SDK de Phonegap)
  • Copiamos el archivo cordova-x.x.x.js al directorio assets/www
  • Actualizamos el proyecto eclipse para que reconstruya el Build path e incluya la nueva librería
